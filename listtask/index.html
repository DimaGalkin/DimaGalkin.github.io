Lists
What is a list?
A list is an immutable data structure that is used to store multiple values. Lists are also indexed, meaning that each element in the list has a unique index that can be used to access the element. Lists are also iterable, meaning that you can iterate through each element in the list. Lists are also heterogenous, meaning that the elements in the list can be of different types. Lists are also dynamic, meaning that the size of the list can change. Lists are also recursive, meaning that a list can contain other lists.
What are common operations on lists?
It is common for lists to have the following operations:
  • isEmpty()
  • length()
  • get(index)
  • set(index, value)
  • add(value)
  • remove(index)
  • contains(value)
